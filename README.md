# RL Methods for LLM Training

This repository contains a collection of **interactive notebooks** to experiment with the main **Reinforcement Learning methods for Large Language Models (LLMs)**.

The project is composed of **Google Colab notebooks** designed to test and compare different RL training methods using different models and datasets.

---
